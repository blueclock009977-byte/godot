## 基準ステータス（再設計）

**1Fで草原雑魚レベル、10Fで森ボスレベルになるように逆算**

### バランス型（2体）
現状: HP55-60, ATK12-14
修正: HP30, ATK6, DEF4, AGI5, MAG0
→ 1F(×0.55): HP16, ATK3 ≒ 草原スライム
→ 10F(×2.8): HP84, ATK17 ≒ 森ボス級

## 実装方針

1. **基準ステータスを下げる**（現在の約50%）
2. **倍率カーブを緩やかに開始、後半急上昇**
3. **敵の数・耐性・スキルで複合的に難易度調整**

## コンセプト別の役割

| コンセプト | 難易度要素 | 対策 |
|-----------|----------|------|
| バランス型 | 基準、数で圧力 | 汎用パーティでOK |
| 物理耐性 | 物理ダメ半減 | 魔法職必須 |
| 魔法耐性 | 魔法ダメ半減 | 物理職必須 |
| 高AGI | 命中しにくい | AGI高いキャラ、必中スキル |
| 高火力 | 被ダメ大 | 高HP、回復、DEF |
| デバフ型 | ATK/AGI低下 | 状態耐性、解除スキル |
| 再生型 | HP削りきれない | 高火力一点集中 |
| 複合耐性 | 物理魔法両方減 | 属性攻撃、貫通 |
| 高HP壁 | 超耐久 | 持久戦、継続火力 |
| ボス | 全部入り | 総合力勝負 |
