# 生物改造機能 企画書 v2

## 概要

キャラクターに「改造枠」を解放し、ボーナスを自由に選択・付与するシステム。

---

## 基本仕様

### 改造枠
- **初期枠数**: 0枠
- **最大枠数**: 15枠
- **解放コスト**: 500コインで3枠解放
- **解放回数**: 最大5回（0→3→6→9→12→15）
- **総コスト**: 2500コイン（500×5回）

### ボーナス選択
- 枠を消費してボーナスを選択
- **いつでも入れ替え可能**（コストなし）
- ボーナス項目は**全キャラ共通**
- **同じボーナスは選択不可**

## 組み合わせ例

### タンク型（15枠）
- HP強化III (1枠)
- DEF強化III (1枠)
- 物理耐性 (3枠)
- 不死身 (10枠)
→ 合計15枠

### アタッカー型（15枠）
- ATK強化III (1枠)
- クリティカル率UP (3枠)
- 物理攻撃UP (3枠)
- HP吸収 (3枠)
- 追撃 (5枠)... ❌枠オーバー

### 火属性特化（15枠）
- MAG強化III (1枠)
- 魔法攻撃UP (3枠)
- 火属性攻撃UP (5枠)
- 会心の極み (10枠)
→ 合計19枠 ❌

→ **15枠だと10枠1個+5枠1個が限界**、ビルドに悩む設計
