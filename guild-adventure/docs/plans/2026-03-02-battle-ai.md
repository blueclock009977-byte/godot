# バトルAI設定機能 企画書

## 概要

キャラクターごとにバトルAIを設定できる機能。
職業ごとにデフォルトパターンが設定されている。

---

## AIパターン（5種類）

### 1. バランス型
- 回復・バフ・デバフ・攻撃を状況に応じて使い分け
- 味方HP低下時は回復優先
- 汎用的なAI

### 2. ブレイク型
- **50%**: デバフを使用
- **50%**: 攻撃（与ダメ最大を75%で選択）
- デバッファー兼アタッカー向け

### 3. アタック型
- **全攻撃手段（通常攻撃 + スキル）から与ダメ最大を選択**
- 75%で1番、25%で2番を選ぶ
- 回復・バフ・デバフは使わない
- 純粋アタッカー向け

### 4. サポート型
- **優先順位**: バフ → デバフ → 回復 → 攻撃
- バフ可能なら使う
- デバフ可能なら使う
- 味方HP低いなら回復
- 最後に攻撃
- バッファー・サブヒーラー向け

### 5. ヒーラー型
- 回復を最優先
- 味方HP70%以下で即回復
- 回復職向け

```typescript
  | 'balanced'  // バランス型
```

```typescript
  // 属性相性（有利なら1.5倍、不利なら0.5倍）
  // 物理耐性 / 魔法耐性
  // 対象数（全体攻撃は合計ダメージ）
}
```

### 考慮する要素
- ✅ **通常攻撃も候補に含む**
- ✅ スキル倍率
- ✅ 属性相性
- ✅ 物理耐性（physicalResist）
- ✅ 魔法耐性（magicResist）
- ✅ 対象数

### 選択確率
- **75%**: 1番威力の高い攻撃（通常攻撃 or スキル）
- **25%**: 2番目に威力の高い攻撃
（2番目がなければ1番を使用）

### 例
敵: 魔法耐性50%

| 攻撃手段 | 期待値 |
|---------|-------|
| 通常攻撃 | 100 |
| メテオ（魔法） | 90 ← 魔法耐性で減少 |
| 斬撃（物理スキル） | 120 |

→ **75%で斬撃、25%で通常攻撃**を選択
→ メテオは魔法耐性で弱いので選ばれない

### 現行との比較
現行: `pickRandom(attackSkills)` ❌ 完全ランダム、通常攻撃と別判定
改善: `selectBestAttack(allOptions, targets)` ✅ 通常攻撃含めて賢い選択
