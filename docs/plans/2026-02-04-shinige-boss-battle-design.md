# 2D死にゲー ボス戦アクション 設計書

## 概要

**ジャンル**: 2D横スクロール・ボス戦特化アクション（死にゲー）

**コンセプト**: 巨大モンスターとの1対1の死闘。何度も死にながらパターンを覚え、強化を重ねて全10体のボスを討伐する。

**プラットフォーム**: PC（Godot Engine）

---

## ゲームの流れ

1. タイトル画面 → メインメニュー
2. メインメニューで「ステージ選択」か「強化」を選ぶ
3. ステージを選んでボスに挑戦
4. 勝利 → G（ゴールド）獲得、新ステージ解放
5. 敗北 → メインメニューに戻る（獲得Gなし）
6. Gを使って武器・回避・生存能力を強化
7. ステージ10クリアでエンディング

---

## 難易度曲線

- **ステージ1**: 攻撃パターンはシンプルだが、回避・攻撃のタイミングを覚えないと勝てない。何度か死んで操作に慣れる想定
- **ステージ2〜5**: パターンが複雑化。複数の攻撃を組み合わせてくる
- **ステージ6〜9**: 高速化・高火力。強化も活用しないと厳しい
- **ステージ10**: 全パターンの集大成。最難関

---

## プレイヤー操作

### 移動
- 左右移動（地上・空中どちらも可）
- ジャンプ（空中の攻撃を避ける、高所の弱点を狙う）

### 攻撃
- **剣（メイン攻撃）**: ボタン押しで斬撃。近づく必要があるがダメージ高い。攻撃後に少し隙がある
- **弓（サブ攻撃）**: 別ボタンで矢を撃つ。離れて当てられる。弾数制限あり（最大5発、時間で回復）

### 回避
- **ダッシュ**: 素早く横に移動。短い無敵時間。連続使用にクールダウンあり
- **ローリング**: 転がり回避。無敵時間が長いが、終わりに隙がある

### 基本の立ち回り
1. ボスの攻撃を見る
2. ダッシュ or ローリング or ジャンプで回避
3. 隙に近づいて剣で斬る
4. 離れて弓で追撃
5. 繰り返し

## 強化システム

### 通貨: G（ゴールド）
- 初期所持: 0G
- ステージクリアで獲得（難しいステージほど多い）
- クリア済みステージも再挑戦でG獲得可能

### 強化項目

| カテゴリ | 強化項目 | 効果 |
|---------|---------|------|
| **攻撃** | 剣ダメージ | 一撃のダメージ増加 |
|  | 弓ダメージ | 矢のダメージ増加 |
|  | 弓弾数 | 最大弾数UP（5→6→7...） |
|  | 攻撃速度 | 剣の振りが速くなる |
| **回避** | ダッシュ距離 | 移動距離が伸びる |
|  | ダッシュ回復 | クールダウン短縮 |
|  | ローリング無敵 | 無敵時間が伸びる |
|  | ジャンプ力 | より高く飛べる |
| **生存** | 最大HP | 体力増加 |
|  | 被ダメ軽減 | 受けるダメージ減少 |

- 各項目にレベル上限あり（Lv1〜Lv5）
- 強化なしでも全クリ可能だが、強化すると楽になる

## ボス・ステージ設計

### 構成
- 全10ステージ、各ステージに1体の固有ボス
- ステージ1クリア → ステージ2解放、という順序
- クリア済みステージは再挑戦可能（G稼ぎ）

### ボス設計の基本
- 各ボスは3〜5種類の攻撃パターンを持つ
- HP残量で行動が変化（後半は激しくなる）
- 攻撃の「予備動作」を見て回避方法を判断する

### ボス案

| ステージ | ボス案 | 特徴 |
|---------|-------|------|
| 1 | スライム系 | 体当たり、ジャンプ攻撃。シンプルだが油断すると死ぬ |
| 2〜3 | 獣系 | 突進、引っかき、咆哮。スピード感ある戦い |
| 4〜6 | 竜・飛行系 | 空からの攻撃、ブレス。ジャンプと弓が活きる |
| 7〜9 | 大型・重量系 | 範囲攻撃、地震、高火力。ローリング必須 |
| 10 | ??? | 全攻撃パターンの集大成。最強 |

## 画面構成

### 画面遷移
```
タイトル画面
    ↓
メインメニュー
    ├── ステージ選択 → ボス戦 → 勝利/敗北 → メインメニューへ
    └── 強化ショップ → 強化購入 → メインメニューへ
```

### タイトル画面
- ゲームロゴ
- 「はじめる」「つづきから」

### メインメニュー
- 所持G表示
- 「ステージ選択」ボタン
- 「強化」ボタン

### ステージ選択画面
- ステージ1〜10を並べて表示
- クリア済み / 未クリア / 未解放 がわかるアイコン
- 獲得G表示（再挑戦の参考用）

### ボス戦画面
- プレイヤーHP（画面左上）
- ボスHP（画面上部、横長バー）
- 弓の残弾数（画面左下）

### 敗北時
- 「YOU DIED」的な表示
- 「リトライ」「メニューに戻る」の選択

Godot Engine

## MVP（最小限の完成形）

1. プレイヤー操作（移動、ジャンプ、剣、弓、ダッシュ、ローリング）
2. ボス1体（ステージ1）との戦闘が成立
3. 勝利 / 敗北の判定
4. メインメニュー ↔ ボス戦の画面遷移

## 拡張順序

1. まず「1体のボスと戦えるプロトタイプ」を作る
2. 手触りが良ければ、強化システムとボスを追加
3. 最後に演出やバランス調整（SE、画面揺れ、ヒットストップ等）
4. セーブ / ロード機能
