# Dice Deck Random v2 — 基礎ルール

## コンセプト
ダイスを使った運ゲーに見せかけて、実はプレイスキルがめっちゃ重要なカードゲーム。

## 基本
- 1vs1
- プレイヤーHP: 20
- デッキ: 20枚（同名カード2枚まで）
- 初期手札: 3枚

## マナ
- 毎ターン最大値+1（上限5）
- ターン開始時に最大値まで全回復
- 用途: 召喚（カードごとにコスト）& 移動（1マナ）

## ターンの流れ
1. **マナ回復**（最大値+1）
2. **メインフェイズ1** — 召喚/移動（マナ消費）
3. 🎲 **ダイスロール** → バトル処理
4. **ドロー1枚**
5. **メインフェイズ2** — 召喚/移動（残りマナ消費）
6. **ターンエンド**

※ マナは1ターン通して共有。メインフェイズ1で使った分はメインフェイズ2では使えない。

## 先行/後攻
- **先行1ターン目**: ダイスなし、ドローなし（メインフェイズ1→メインフェイズ2→エンド）
- **後攻1ターン目**: 通常通り（ダイスあり、ドローあり）

## フィールド（3×2、対面制）
```
     [O4][O5][O6]  ← 相手後列
     [O1][O2][O3]  ← 相手前列
     ─────────────
     [P1][P2][P3]  ← 自分前列
     [P4][P5][P6]  ← 自分後列
```

### レーン
- 左レーン: P1/P4 ↔ O1/O4
- 中レーン: P2/P5 ↔ O2/O5
- 右レーン: P3/P6 ↔ O3/O6

### 前列/後列
- ボーナスなし
- 後列カードは同レーンの味方前列がいる限り攻撃されない
- 後列カードも攻撃可能（同レーンの相手前列を殴る）

## ダイス & バトル処理

### ダイス
- 6面ダイス1個を振る
- **両者共有** — 両陣営のカードがこの出目で反応

### カードの攻撃ダイス
- カードごとに「攻撃ダイス」を持つ（例: [1,3,5]）
- 振ったダイスの目がカードの攻撃ダイスに含まれていれば、そのカードは攻撃する

### バトル処理順
1. **ターンプレイヤー**の該当カードが 1→2→3→4→5→6 の順に**1体ずつ**攻撃
2. **生き残った相手**の該当カードが 1→2→3→4→5→6 の順に**1体ずつ**攻撃

### 攻撃対象（自動・同レーン内）
優先順: **相手前列 → 相手後列 → 相手本体**

- 同レーンの相手前列にカードがいれば、そのカードを攻撃
- 相手前列が空なら、同レーンの相手後列を攻撃
- 両方空なら、相手プレイヤー本体にダメージ

## 勝敗条件
- 相手HPを0以下にしたら勝利
- デッキ切れ: ドローできない（ゲーム続行）

## カードステータス設計

### ポイント制
各コストに総ポイント予算を設定し、ATK/HP/ダイス/効果に振り分ける。

| コスト | 総ポイント |
|---|---|
| 1 | 6pt |
| 2 | 9pt |
| 3 | 12pt |
| 4 | 15pt |
| 5 | 18pt |

### ポイント配分

| 要素 | コスト |
|---|---|
| ATK +1 | 1pt |
| HP +1 | 1pt |
| 攻撃ダイス +1個 | 1pt |
| 効果 | 2-4pt（強さによる） |

### 同コスト内バリエーション
- **ステ特化型**: ATK/HPにポイントを集中。硬い壁や高火力
- **ダイス特化型**: 攻撃ダイス多め。毎ターン動ける小回り型
- **効果持ち**: スペック低めだが特殊効果で戦局を変える
- **バランス型**: 平均的に振り分け
