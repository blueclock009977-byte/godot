# Idle Dungeon - ゲーム企画書

## コンセプト
**「見下ろし型2D放置ハクスラRPG」**

Archero風のビジュアル × 完全放置 × ハクスラ装備

## ゲームループ
```
1. アプリを開く
2. 放置中の進行結果を確認
   - 「32階まで進んだ！」
   - 「装備3個、スキル書2個ドロップ！」
3. 装備・スキルを付け替え
4. アプリを閉じる（放置再開）
```

## 基本仕様

### キャラクター
- **1キャラ固定**（作成不要、最初から存在）
- ステータス: HP, ATK, DEF, SPD
- レベルアップで自動成長

### 装備システム
- **武器**: ATK上昇
- **防具**: DEF上昇
- **アクセサリ**: 特殊効果
- レアリティ: Common / Rare / Epic / Legendary
- 同じ装備を合成して強化可能

### スキルシステム
- **パッシブスキル**のみ（操作不要）
- 例: 「クリティカル率+10%」「HP自動回復」
- スキルスロット: 最大4つ

### ダンジョン（階層）
- 1階から開始
- 各階に敵が出現
- 倒すと次の階へ
- 5階ごとにボス
- 死亡 → その階からやり直し

### 放置システム
- オフライン中も自動進行
- 最大8時間分まで蓄積
- 再開時に結果をまとめて表示

### ランキング
- 最高到達階層でランキング
- グローバルランキング

## データ構造

### ユーザーデータ（Firebase）
```typescript
interface UserData {
  username: string;
  character: {
    level: number;
    exp: number;
    hp: number;
    atk: number;
    def: number;
    spd: number;
  };
  currentFloor: number;
  highestFloor: number;
  equipment: {
    weapon: string | null;
    armor: string | null;
    accessory: string | null;
  };
  inventory: string[];  // 装備ID配列
  skills: string[];     // スキルID配列（最大4）
  skillInventory: string[];  // 所持スキル
  lastActiveAt: number; // 最終アクティブ時刻
}
```
